ฉันอยากได้เว็บโทนสีฟ้า และสีตรงข้ามกับสีฟ้า อยากให้เน้น Contrast 
สไตล์เรียบ คุณภาพ แต่ทันสมัย 
ฟอนต์ที่ต้องการคือ Prompt ทั้งเว็บ โดย Import มาใช้ทุกน้ำหนักเลย

ออกแบบหน้าตาเว็บให้มี Design System เดียวกันทั้งเว็บ
- ปุ่มทุกหน้าต้องใช้สไตล์เดียวกัน
- การ์ดทัวร์ การ์ดโรงแรม รูปแบบเหมือนกัน
- มุมโค้ง ระยะห่าง และเงา ต้องสม่ำเสมอ
- สไตล์เรียบ ดูมีคุณภาพ ไม่รก

ออกแบบ UX โดยเน้นให้ผู้ใช้งานจองได้ง่ายและรวดเร็ว
ลำดับการใช้งานควรเป็น:
หน้าแรก → เลือกบริการ → ดูรายละเอียด → กรอกข้อมูลจอง → แสดงผลการจอง

ออกแบบเว็บแบบ Mobile-first
- บนมือถือ การ์ดแสดง 1 คอลัมน์
- ปุ่มต้องใหญ่ กดง่าย อ่านชัด
- ปุ่มจองควรอยู่ในตำแหน่งที่เข้าถึงง่ายบนมือถือ
- บนจอใหญ่ค่อยปรับเป็นหลายคอลัมน์

เขียนโค้ดด้วย React โดยแยกโครงสร้างชัดเจน
- แยกหน้า (pages)
- แยกส่วนที่ใช้ซ้ำได้ เช่น ปุ่ม การ์ด เมนู
- โค้ดอ่านง่าย แก้ไขต่อได้ในอนาคต
- ใช้ Tailwind CSS เป็นหลัก
- เซ็ททุกอย่างที่เป็น Tailwind CSS ไว้สักไฟล์ดึง เวลาจะแก้ดีไซน์หรือ CI Brand จะได้มาแก้ตรงนี้ แก้ง่าย กำหนดสี ฟอนต์ ขนาด ระยะห่าง และสไตล์หลักไว้ที่ส่วนกลาง เพื่อให้เปลี่ยน CI Brand ได้ง่ายในอนาคต

ข้อความบนเว็บไซต์ใช้โทนมืออาชีพ น่าเชื่อถือ
เหมาะกับธุรกิจท่องเที่ยวและบริการ
ไม่ใช้ภาษาขายของเกินจริง
อ่านง่าย เป็นมิตรกับนักท่องเที่ยว
อ่านแล้วเข้าใจเลย ชักจูง แต่ไม่เกินจริง

ใช้โทนสีฟ้าเป็นสีหลักของเว็บไซต์
ใช้สีตรงข้ามกับสีฟ้า (cyan) เป็นสีเน้น (Accent) เพื่อให้เกิด Contrast ชัดเจน
พื้นหลังเรียบ สีอ่อน ตัวอักษรอ่านง่าย
เน้นความสะอาด ทันสมัย และดูมีคุณภาพ

ใช้ฟอนต์ “Prompt” ทั้งเว็บไซต์
นำเข้าและใช้งานทุกน้ำหนักของฟอนต์
กำหนดลำดับขนาดตัวอักษรให้ชัดเจน
เช่น หัวข้อใหญ่ / หัวข้อย่อย / เนื้อหา
ให้ใช้ตามน้ำหนักฟอนต์ที่เอาเข้ามา ไม่ต้องใช้ของ CSS
